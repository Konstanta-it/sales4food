# Ведомость расчетов с поставщиками

Отчет **"Ведомость расчетов с поставщиками"** выводит информацию о расчетах организации с поставщиками за определенный период.

Отчет расположен в разделе **"Деньги"** - **Отчеты**

В колонке 
- "Начальный остаток" выводятся итоги расчетов на начало периода  
- "Приход" выводятся суммы долга, сформировавшиеся в период отчета
- "Расход" выводятся суммы оплаты за период отчета
- "Конечный остаток" выводятся итоги расчетов на конец периода
