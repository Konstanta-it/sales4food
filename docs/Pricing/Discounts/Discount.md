# Скидки (наценки)

Документ **"Скидка (наценка)"** предназначен для описания скидок / наценок, применяемых к Соглашениям с контрагентами.   
Список документов расположен в разделе **"Заказы"** в подсистеме **"Цены и скидки"** - **"Скидки наценки"**.

[![1][1]][1]

При создании документа **"Скидка (наценка)"** указываются:

- **Наименование**
- **Номер** и **дата** (заполняются автоматически)
- **Период действия скидки**
- Дата досрочного завершения скидки (если скидку необоходимо заверишь раньше конца периода действия)
- **Согласование** Черновик, Ожидает согласования, Согласован, Отменен. Только документ со статусом Согласован будет действовать в системе   
- Маркетинговое мероприятие (заполняется автоматически при создании документа на основании из маркетингового мероприятия)
- **Тип** Процентом на номенклатуру, Фиксированная цена, Процентом на товарную категорию
- Признак Уточнять до характеристик (доступно для типов Процентом на номенклатуру и Фиксированная цена) позволяет на странице "Номенклатура" указывать характеритсику номенклутуры 
- **Тип объекта** Соглашение с контрагентом, Точка доставки

**Вкладка "Объект скидки"** 

Указываются соглашения с контрагентами / точки доставки, к которым будет применяться скидка / наценка в зависимости от типа объекта скидки.

![2]

**Вкладка "Номенклатура" / "Товарная категория"** 

Вкладка заполняется в зависимости от выбранного типа скидки/наценки:

- **"Процентом на номенклатуру"**. Определяется список номенклатур, на которые назначаются скидки, а так же размер скидки для каждой номенклатуры в процентах (в случае наценки задается отрицательное число).

- **"Фиксированная цена"**. Определяется список номенклатур, на которые будут назначаться цены, а так же устанавливаются цены в выбранной валюте.

- **"Процентом на товарную категорию"**. Определяется список товарных категорий, на которые назначаются скидки, а так же размер скидки для каждой товарной категории в процентах (в случае наценки задается отрицательное число).

# Промо объем

Для типов скидки "Процентом на номенклатуру" и "Фиксированная цена" доступен ввод **промо объема**. 

!!! attention ""  

    Промо объем - это дополнительное условие к документу помимо периода и соглашения.  
    Скидочная цена устанавливается на определенный объем продукции.   

Промо объем уменьшается с заказами клиентов, в которых участвует скидка, реализациями товаров и корретировками в большую сторону.  
Расход уменьшается с корректировками в меньшую сторону и возвратами товаров от клиентов.   
Расход в заказе как бы бронирует некоторое количество промо, которое потом снимается с созданием реализации по заказу.   
Отслеживать остаток промо объема удобно отчетом "Остатки промо объемов"
