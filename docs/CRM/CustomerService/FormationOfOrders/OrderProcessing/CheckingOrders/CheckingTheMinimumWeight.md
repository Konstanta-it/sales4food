# Проверка минимального веса

В разделе **"Заказы"** в **"Настройках исполнения заказов"** в **"Проверках документов"** устанавливаем проверку - **"Минимальный вес"** для документов **Заказ клиента**.

Проверка сравнивает установленный в соглашении минимальный вес заказа с весом в заказе клиента

![1]

Если проверка не прошла, то появляется информационное сообщение с причиной непрохождения проверки
