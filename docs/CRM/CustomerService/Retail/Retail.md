# Розничная продажа

В системе есть раздел "Розничная продажа", который используется для загрузки отчетов о рознице  

Для включения розничных объектов нужно включить настройку "Использовать розничную продажу" в подсистеме "Заказы" в "Настройки исполнения заказов"

![1]

После включения настройки в подсистеме "Заказы" станет доступен раздел "Розничная продажа"

В разделе доступны следующие справочники :  

- Виды оплат (используется для заполнения данных в справочнике "Эквайринговые терминалы")   

![2]

- Кассы ККМ  

![3]

- Эквайринговые терминалы  

![4]  
![5]
