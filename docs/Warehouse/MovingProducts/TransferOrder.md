# Формирование заказов на перемещение

Документ **"Заказ на перемещение"** отражает потребность в перемещении с одного склада предприятия на другой. Они расположены в разделе **"Заказы"**.

[![1][1]][1]

!!! attention ""  

    Для пользователей, которые занимаются созданием заказов клиентов в системе создан поставляемый профиль доступа "Создание заказов на перемещение".  

При создании документа автоматически подставляются статус документа и этап (если используются в системе).

На вкладке **"Основное"** указываются:

- **Номер документа** (заполняется автоматически)
- **Дата** (дата создания документа в системе)
- **Организация** (видна, если в системе несколько организаций. Заполняется автоматически, если указана основная организация)
- **Склад отправитель**
- **Склад получатель**
- **Тип доставки** Не требуется \ Доставка сторонней транспортной компанией \ Доставка нашей транспортной компанией. Заполняется из карточки склада получателя, можно заполнить вручную  
- Информация о дате отгрузки и дате и времени доставки.  
Даты рассчитываются автоматически при заполнении одной из них с учетом количества дней доставки, указанного в складе получателе.   
Время доставки заполняется из склада получателя, можно заполнить вручную 

[![2]][2]

**Вкладка "Товары"**

На вкладке Товары указывется список заказываемой номенклатуры.  
Заполняются номенклатура, её характеристика, количество упаковок, упаковка (колонка Ед.).   

[![3]][3]

**Вкладка "Дополнительно"**  

На вкладке Дополнительно автоматически заполняется способ создания заказа и ответственный (пользователь, который создал заказ в системе).  

Подразделение заполняется автоматически, если указано основное подразделение.

# Загрузка заказов из файлов

Заказы клиентов могут быть загружены из Excel (из файла или бланка).

[![4][4]][4]

Бланки для заказов создаются в АРМе *"Создание бланков заказов"*, который расположен в разделе **"Заказы"**.

[![5][5]][5]

На вкладке **"Заказы на перемещение"** формируются бланки для заказов на перемещение товаров. Заполняются склад отправитель, склад получатель, каталог, куда будет выгружен бланк, признак отправки на почту и почтовый адрес.

[![7][7]][7]

На вкладке **"Настройки"** заполняются дата (планируемая дата доставки заказа), организация, подразделение. Признак **защищать бланки** позволяет защитить от редактирования все колонки таблицы товаров выгружаемого бланка кроме количества и даты доставки.

**Настройки каталога товаров** включают в себя условия на товары, которые будут указаны в бланке:

- Только с ценой. В бланке будут указаны только те товары, на которые установлена цена на дату доставки по виду цены из соглашения с контрагентом  
- Доступные по сезонам. В бланке будут указаны товары, для которых не установлен сезон или для которых дата доставки попадает в сезон  
- Доступные для продажи. В бланке окажется номенклатура для которой дата доставки попадает в диапазон между датами ввода в продажу и вывода из продажи

[![6][6]][6]

В таблице товаров указывается количество заказываемых товаров.

[![8][8]][8]

При загрузке бланка указывается каталог, из которого будет загружен бланк и каталог, в который этот бланк будет перемещен после загрузки.

[![9][9]][9]

При загрузке создается заказ на перемещение.

Для **"Настроек загрузки заказов"** заполняются:

- Наименование
- Признак автоматической загрузки 
- Признак создания одного заказа на дату. Если признак включен, то при загрузке бланка с той же датой отгрузки, точкой доставки и соглашением повторно данные в старом заказе будут изменены на загружаемые. Если признак выключен, то будут формироваться новые заказы  
- Путь к папке загрузки
- Путь к папке обработанных бланков

При запуске регламентного задания **"Загрузка заказов из бланков"** в систему загружаются заказы на перемещение согласно настройкам.

# Настройки исполнения заказов

В разделе **Заказы** в подсистеме **"Сервис"** - **"Настройки исполнения заказов"** можно включить дополнительные настройки создания заказов.

**"Контроль остатков заказов"** не позволяет после оформления по заказу плана отгрузки сделать количество товаров в заказе клиента меньше чем в плане.   
При этом количество в плане отгрузки нельзя сделать больше чем в заказе.

!!! attention ""  

    Настройка распространяется на все виды заказов.
