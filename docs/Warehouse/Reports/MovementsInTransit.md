# Отчет Перемещения товаров в пути

Отчет используется для отслеживания отгрузки по заказу на перемещение и его приемки на складе-получателе  

Отчет находится в подсистеме Заказы - Аналитика - Обработка заказов

Отслеживание происходит для складов, у которых включена настройка Отслеживать товары в пути

Отгружено – количество из распоряжения на отгрузку по заказу на перемещение  
Принято – количество из распоряжения на приемку по перемещению товаров (по заказу на перемещение)
